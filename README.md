# form elements

```
<form>
    <input type="text">             tipo texto
    <input type="password">         tipo senha 
    <input type="email">            tipo email 
    <input type="submit">           para submeter formulario
    <input type="file">             para inserir imagem
    <input type="date">             para escolher uma data no calendario
    <input type="radio">            radio button
    <input type="range">            escolher range

    <textarea name="name" rows="8" cols="80">                   rows significa o tamanho e cols a largura

</form>

name="valor"                        identifica o dado de cada input
```
